## **窗口操作**

### **基础**

* ⌘ Command + W：关闭窗口或标签页  
* ⌘ Command + Q：退出应用  
* ⌘ Command + T：新建标签页  
* ⌘ Command + N：新建窗口  
* ⌘ Command + Tab：切换应用  
* ⌘ Command + ,：打开偏好设置  

<br />

### **进阶**

* ⌘ Command + `：切换同应用的窗口  
* Fn + H：隐藏当前应用窗口  
* Fn + F：全屏/退出全屏  
* Fn + Control + 方向键：调整窗口位置  
* Fn + Shift + Control + 方向键：切换显示器  

<br />

## **截屏操作**

* ⌘ Command + Shift + 3：全屏截图  
* ⌘ Command + Shift + 4：区域截图  
* ⌘ Command + Shift + 4 + Space：窗口截图  
* ⌘ Command + Shift + 5：打开截图工具栏  
* 加 ⌥ Option：复制到剪贴板（不保存到桌面）  

<br />

## **Dock 栏**

* 按住 ⌘ Command 点击：打开文件位置  
* Fn + A：切换 Dock 自动隐藏  

<br />

## **其他功能**

* ⇧ Shift + ⌥ Option + 音量键：微调音量  
* Fn + E：弹出光盘（仅限光驱机型）  
* Fn + C：打开控制中心  
* Fn + Q：打开快速备忘录  
* Fn + N：新建备忘录  
* Fn + Delete：在回收站中还原文件  
* ⌥ Option + ⌘ Command + Esc：强制退出应用  

<br />

## **文本编辑**

* ⌘ Command + 方向键：跳转到行首/行尾  
* ⇧ Shift + 方向键：多选文本  
* Fn + Delete：向前删除（相当于 Windows 的 Delete）  
* Fn + ↑：Page Up（向上翻页）  
* Fn + ↓：Page Down（向下翻页）  
* Fn + ←：Home（跳转到开头）  
* Fn + →：End（跳转到末尾）  
* Control + A：移动到行首  
* Control + E：移动到行尾  
* ⌥ Option + ←：跳转到前一个单词  
* ⌥ Option + →：跳转到后一个单词  

## **Finder 操作**

* 按住 ⌘ Command 双击：在新窗口打开文件夹  
* ⌘ Command + K：连接服务器  
* ⌘ Command + ⇧ Shift + .：显示/隐藏隐藏文件  
* ⌘ Command + ⇧ Shift + G：前往文件夹  
* ⌘ Command + ⇧ Shift + N：新建文件夹  
* ⌘ Command + Control + N：为选中文件创建文件夹
* ⌘ Command + ↑：返回上层文件夹  
* ⌘ Command + ↓/O：打开选中的文件  
* ⌘ Command + I：显示简介（可修改默认打开方式和图标）
