ESP8266 Web Serial Flasher

选择 PlatformIO 生成的 .pio\build\nodemcuv2\firmware.bin，通过 Chrome/Edge 的 Web Serial 烧录 ESP8266。


    


    

      
使用步骤

      
1. 先在 PlatformIO 里编译 ESP8266 工程，得到 .pio\build\nodemcuv2\firmware.bin。

      
2. 用 USB 连接 ESP8266。如果自动下载失败，按住 BOOT/FLASH，让 GPIO0 接地后复位进入下载模式。

      
3. 点击“选择串口”，再选择 firmware.bin，点击“烧录固件”。
